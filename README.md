## 🚀 Features

- **Monthly Reach Intelligence**: Specifically tracks "this month" performance (30-day window) as requested in the founder's napkin sketch.
- **Keyless Architecture**: Uses `youtube-sr` to bypass Official Google API quota limits.
- **Deep KPI Dashboard**: Instantly calculates Average Views, Top Video, and Monthly Reach.
- **Temporal Indicators**: "NEW" badges for videos posted this month and Trending icons (🔥) for high-velocity outlyers.
- **Interactive Visualizations**: High-end performance charts built with Recharts.
- **Data Portability**: One-click CSV export for competitive research.
- **Premium UX**: Dark-mode SaaS aesthetic with loading skeletons and responsive layouts.

---

## 🛠 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS + shadcn/ui
- **Charts**: Recharts
- **Data**: youtube-sr (Internal Scraping API)
- **Icons**: Lucide React
- **Deployment**: Vercel

## 📦 Local Setup

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. **Open [http://localhost:3000](http://localhost:3000)** in your browser.
